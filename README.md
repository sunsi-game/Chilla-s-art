## Overview
이 프로젝트는 언리얼 엔진 기반 공포게임 프로토타입 포트폴리오입니다.  
학교 공간을 배경으로 상호작용, 이벤트 연출, UI 표시, 퍼즐 진행, 사운드 처리 등을 구현했습니다.

## Repository Guide
용량이 큰 외부 에셋(Fab 다운로드 리소스, 고해상도 텍스처, 일부 메쉬 파일)은 저장소에서

 제외했습니다.

대신 프로젝트에서 직접 구현한 핵심 요소들은 확인할 수 있도록 유지했습니다.

- 플레이어 상호작용 시스템
- 문, 엘리베이터, 포털 등 오브젝트 이벤트 처리
- UI 표시 및 컷신 흐름 제어
- 입력 제한 및 상태 전환 처리
- 발소리, 효과음, 배경음 등 사운드 연동
- 블루프린트 기반 기능 분리 및 디버깅 과정

프로젝트의 실제 동작 결과는 아래 시연 영상과 설명 자료를 통해 확인할 수 있습니다.
